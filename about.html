About This Project

Project Overview

By highlighting variations in walk-based access to public transportation across neighborhoods, this project gives users an intuitive way to compare accessibility levels in different parts of Seattle. It provides a clear visual explanation of how public transportation connects different parts of Seattle and how it intersects with daily mobility.

For this project, we focused on creating an interactive smart dashboard about transportation accessibility in the Seattle area. Our intended audience includes students, elders, commuters, transit-dependent residents, and people without access to private vehicles.

Project Functions

This webpage is an interactive smart dashboard that visualizes public transit accessibility across Seattle. Users can select a neighborhood or view the entire city to see the nearest bus stops, along with viewing filters and a "Neighborhood Accessibility Index" to calculate the transportation accessibility of the selected neighborhood using the number of stops and the neighborhood land area.

The dashboard supports comparison between neighborhoods and helps identify areas with strong transit access as well as areas that may be underserved.

Data Sources

Data used in this map include King County Metro transit stops (King County, Washington), Atlas Neighborhood boundaries (City of Seattle ArcGIS Online), and transit route datasets from the Seattle Transportation Plan (City of Seattle ArcGIS Online).
