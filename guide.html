Public Transportation In Seattle

What is this interactive map for?

This project visualizes public transit services in Seattle using interactive web mapping. The goal is to help users explore transit routes, stop accessibility, and neighborhood-level transportation patterns across the city.

How can this map help me?

  • I am a student or elder
  • I don't have access to a personal vehicle
  • I don't have a drivers license
  • I am a commuter
  • I rely on public transportation
  • I need to find accessible routes
  • I want to compare transit stops in neighborhoods
  • I am planning a trip

How do I use this map?

Features

  • "Zoom to a (Seattle) Neighborhood" selection
  • "# Transit Stops in the Displaying Region" (dynamic chart)
  • Compute "Neighborhood Accessibility Index"
  • Hide dashboard
  • Reset dashboard
  • Fiter Layers
  • Map zoom buttons

Legend

  • North Bound Transit Stops
  • South Bound Transit Stops
  • East Bound Transit Stops
  • West Bound Transit Stops
  • Routes
  • Neighborhood Boundaries
